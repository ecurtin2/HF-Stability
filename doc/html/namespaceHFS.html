◆ davidsonAgreesWithFullDiag()

bool HFS::davidsonAgreesWithFullDiag ( )

Checks that Davidson's Algorithm is getting the lowest eigenvalue.

The full matrix is built and diagonalized, which is slow for Nk > 20 or so. The minimum of the eigenvalues is compared to the value returned by Davidson's algorithm and is said to be equal if they are the same within SMALLNUMBER

Returns
true if |DavidsonValue - MinimumEigenvalue| < SMALLNUMBER, otherwise false
See also
buildMatrixFromFunction()
